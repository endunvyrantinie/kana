# Kanun

Malaysian HR & Employment Law, played in 60 seconds.

A gamified micro-learning platform covering the **Employment Act 1955**, the **Sabah Labour Ordinance (Cap. 67)** and the **Sarawak Labour Ordinance (Cap. 76)**.

## What's in this repo

- `index.html` — single-file playable demo (six game modes, XP/level system, paywall). No build step. Open in a browser or deploy to any static host.
- `BLUEPRINT.md` — full product blueprint: game modes, XP curve, paywall psychology, content engine, DB schema, UX flows, MVP scope.

## Local preview

Just open `index.html` in your browser. Or:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Deployment

Static. Drop into Vercel / Cloudflare Pages / Netlify — no build command needed.
